Task: Simple Contact Manager (Console)

Write a C program that:

Defines a struct Contact with:

char name[50]

char phone[20]

Allows the user to:

Add up to 5 contacts

Display all saved contacts

Program flow:

Ask the user how many contacts they want to enter (max 5)

For each contact:

Ask for name

Ask for phone number

After input, print all contacts in a clean format

Requirements (important)

Use functions (don’t do everything in main)

e.g. addContacts(...), printContacts(...)

No global variables

Handle input safely (no gets)

Keep it simple — no dynamic memory yet
